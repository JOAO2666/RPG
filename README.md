## Estrutura do Projeto
- `src/` - Código fonte do jogo
- `include/` - Arquivos de cabeçalho
- `assets/` - Recursos do jogo (imagens, sons, etc.)

## Requisitos
- Compilador C/C++
- Biblioteca SDL2 (para gráficos)
- Git

## Como Compilar
```bash
gcc -o rpg src/*.c -I include -lSDL2
```
